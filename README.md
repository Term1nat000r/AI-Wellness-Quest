# AI Wellness Quest

**Веб-приложение для трекинга здоровья с AI-мотивацией и социальной поддержкой**

Платформа для отслеживания здоровых привычек, где пользователи получают персонализированные челленджи от ИИ, участвуют в рейтинге и общаются в сообществе. Backend на Flask + SQLite, фронтенд на чистом HTML/CSS/JS, AI-генерация через GigaChat API.

> Проект создан в рамках учебной задачи по архитектуре программного обеспечения

## Архитектура

### Уровень 1 — Контекстная диаграмма

Показывает, как приложение взаимодействует с внешним миром.

| Элемент | Описание |
|---------|----------|
| Пользователь | Человек, который следит за здоровьем |
| AI Wellness Quest | Основное веб-приложение |
| GigaChat API | Внешний AI-сервис Сбера |

```mermaid
C4Context
    title Уровень 1 — Контекстная диаграмма

    Person(user, "Пользователь", "Человек, который следит за здоровьем")

    System(wellness, "AI Wellness Quest", "Веб-приложение для трекинга здоровья и AI-мотивации")

    System_Ext(gigachat, "GigaChat API", "Внешний AI-сервис от Сбера")

    Rel(user, wellness, "Использует через браузер")
    Rel(wellness, gigachat, "Запрашивает генерацию контента")

    UpdateLayoutConfig($c4ShapeInRow="3", $c4BoundaryInRow="1")
